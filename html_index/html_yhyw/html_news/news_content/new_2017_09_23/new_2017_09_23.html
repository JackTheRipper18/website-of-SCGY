2017新生篮球小组赛顺利结束

发布时间:2017-9-23  图文:落樱 编辑:思归人

  刚入学的新同学们想必都想在新校园里大展身手（搞一波事）吧，于是，以增进同学友谊为目的，展现提升个人能力为方向，学长们看戏为附加（？？？），2017年新生篮球及足球班赛正式开始啦。

   篮赛将少院六个班分为AB两组，第一轮进行组内互赛，每组淘汰一个班。第二轮半决赛，将由A组第一名对战B组第二名，B组第一名对战A组第二名。胜者再进行决赛决出最后的冠军。

  第一战9月13日，班赛第一场正式开始，A组中创新1班与创新3班实力相当，前期创1一直领先于创3，在后半场时创3扭转局势，成功将比分反转，最后创3以25：23的比分翻盘获胜。而另一边B组的001和002的比赛中，001实力上压制002，最终001以35：16的比分获胜。

  第二战AB组的第二场分别在17号和16号开始。16号002与创2的比赛气氛十分紧张，双方前期势均力敌，场上弥漫着一丝火药味，最终创2以25：17战胜002。次日B组的比赛局势就呈一边倒了，创3在上半场就取得了领先少班20分的优势，后期少班虽有还击但力度欠佳最终创3以58：30获胜。

  第三战最后的第三场决胜局于9月21日进行，A组的对决中少班并没有改变被动局面，在维持平分10分钟后就全面崩盘，最终创1以75：19的绝对优势战胜少班。而B组001和创2的比分相当焦灼，双方交替领先，最终后半场创2把握对方的防守漏洞（或许是因为看球妹子比较多？？？）最终以38：33战胜001以小组第一入围。

  经过一个星期的激烈比赛，第一轮初赛已经结束了。通过小编参赛的亲身经历，各位老哥为了班级的荣誉也是都拼了命的奋勇比赛，一个比一个能打能扛能得分。现场紧张的气氛和焦灼的比分完全不输正规比赛。（来自少班的小编看到比分也不禁默默的留下了眼泪......）

  当然，比赛的两班球场虽上是对手，球场下也建立了深厚的友谊，班赛不仅增进了班内的团结合作，更给了不同班级间了解交流的机会，正所谓“不打不相识”，相信通过班赛，可以让少院的同学们更加熟悉，更加团结。

  第二轮的半决赛即将开始，让我们共同期待同学们更加精彩的表现吧！
