物理专业交流会

发布时间:2018-03-20  编辑：网络媒体部 宇燕

   本次交流会由少年班学院学生会承办，孙树文同学主持，并请来了三位少院转物院的的学长为我们介绍经验，他们分别是李嘉伟、胡仲强、夏立乔。

李嘉伟

  首先是李嘉伟学长为我们介绍物理的一些学科特点，让我们对物理有了更加深入的了解，激发了我们对物理的兴趣，也让我们对严济慈科技英才班满怀憧憬。

1. 物理是什么？物理学是旨在揭示物质的最基本的结构及其运动的一般规律。
2. 物理所研究的尺度是从微小粒子到宏观宇宙。
3. 物理学的专业分类有众多方向。
4. 凝聚态物理前沿:详述了 曾杰教授 的一些研究成果，介绍了物理和化学的结合。
5. 严济慈物理科技英才班旨在培养在物理及相关研究领域的领军学者，并且拥有众多福利，以及严班的滚动情况。

胡仲强

  接下来是胡仲强学长为我们介绍科大物理学院的状况。物理学院的方向众多，资源也及其丰富。同时他也介绍了一些他个人的学习方法。

1. 物理整体介绍我们要结合自身，对未来有良好的规划，利用身边的资源，无惧激烈的竞争，并且正视GPA。
2. 细分方向重点介绍了凝聚态物理，并建议我们早进实验室。
3. 学习方法五点注意：选课慎重，听课无害，作业有益，乐者刷题，淡然考试。
4. 总结对大家的一些建议。

夏立乔

  最后，夏立乔学长为我们阐述了：为什么我们选择物理。因为学过物理竞赛，因为数学太难了，因为物理有趣，Because we can!

1.为什么选择物理竞赛≠科研，选择物理专业≠从事物理学研究。
2.物理专业培养的核心能力如：建立模型，归纳关联，数学推导计算，编程，实验。
3.学物理可以干什么？科研，工业，金融，创业。
4.物理学研究什么？分为原子以下，原子以上
5.科研，我能行吗？科研精神回报丰厚，需要长时间对研究领域保持热情，热情=兴趣+获得成果的正反馈。
6.关于物理学习理论：阅读思考远比刷题重要，不要让繁琐的数学掩盖背后的物理，请认真听讲，独立完成作业。实验：预习时关注实验操作，在实验报告中体现思考。

提问

   最后的提问环节，同学们向学长提出了自己的问题，学长们也热情回答了同学的问题。
