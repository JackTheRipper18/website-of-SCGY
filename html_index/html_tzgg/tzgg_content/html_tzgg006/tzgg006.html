专业交流会|信息和计算机

这周六（4月14日）的下午14：00，我们将在东区二教2104教室，邀请三位学长一位学姐介绍信息科学技术学院和计算机科学与技术学院，帮助还没有选择专业 的小伙伴们更加了解这两个学院。

让我们来认识一下他们吧—

符气鸿

2014级网络信息安全系，曾担任广播社首任社长，曾获电磁学小论文一等奖，曾担任量子信息助教，未来计划到英国或新加坡深造

尹童欣

15级少年班学院创新二班团支书，计算机与信息科技英才班，专业方向为6系人工智能，曾获2017国家奖学金、2017英才班培养计划助学金B。

何纪言

来自重庆（Gnome 3.28），16级计算机学院人气最高的同学之一，科大程序在线评测系统的主要开发者，国内高校最强计算机技术社团 USTC LUG 的副社长，计算机学院科技部部长，为科大 Nebula 战队（CTF 竞赛）和鸿雁队（超算竞赛）的发展做出了很大贡献，在国际大学生程序设计竞赛亚洲赛区总决赛中勇夺银牌，喜欢在玩游戏时顺便反汇编修正游戏的 bug。

曾明亮

来自广东，16级学长，入学后从信院通过入学考试转入计算机学院，目前在计算机学院科技部担任核心职务。数理基础扎实，在国际大学生编程竞赛（ICPC）和中国大学生编程竞赛（CCPC）中获得了众多奖项。爱好众多，涉猎广泛，计算机，信息，数学，物理，生物，化学，音乐等学科知识融汇贯通，参加过各学科竞赛，曾经在 iGEM 2017 中负责合成生物学部分。


周六（4月14日）的下午14：00，东区二教2104，菌菌与你不见不散哦~
